# ⚙️ Движок: Present Perfect Continuous

**Формула:** have / has + been + V-ing

**Суть своими словами:** Процесс запустился в прошлом, крутится до сих пор, и нам капец как важно подчеркнуть **длительность** (сколько времени этот счетчик уже тикает). Главные слова-маркеры: *for* (в течение) и *since* (начиная с).

### 🖥️ Примеры из IT / Жизни:
1. *I have been learning Go for 6 months.* (Учу Гошку уже полгода и продолжаю).
2. *My laptop has been making weird noises since yesterday.* (Ноут издает странные звуки со вчерашнего дня).
3. *We have been debugging this server for 3 hours.* (Мы дебажим этот сервер уже три часа).
