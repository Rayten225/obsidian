# ⚙️ Движок: Future Simple

**Формула:** will + V
**Суть своими словами:** Факт в будущем хз тут все просто 

### 🖥️ Примеры из IT / Жизни:
1. *I will do this task* (Я выполню эту задачу)
2. *She will write this abstract* (Она напишет этот конспект)
3. *Will we work tommorow?* (Мы будем работать завтра?)
