# 🪲 Баг: I code in Python for 3 years

**Дата:** 2026-06-22
**Движок (Правило):** [[Present Perfect Continuous]]

**Ляпнул::** "I code in Python for three years"
**Надо::** "I have been coding in Python for three years"

### Анализ сбоя (почему так вышло):
- Чистая калька с русского. В русском языке «я кожу» — это настоящее время. Мозг автоматически скастил это в `Present Simple` (*I code*). 
- Но в английском `Present Simple` со счетчиком времени (*for 3 years*) означает бред: *«Раз в три года я сажусь и немножко кожу»*. 
- Если действие началось в 2023-м и идет сегодня — обязан включиться тяжелый счетчик `have been V-ing`.
