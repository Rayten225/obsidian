
## Интерфейс
---
***Интерфейс*** — это тип данных который описывает набор методов (функций). Отвечает на вопрос ''Что может сделать"

```Go
type Stringer interface {
    String() string
}
```

## Обработка ошибок
---

Чтобы создать ошибку можно воспользоваться 
```Go 
fmt.Errorf("got %d", i)
```
Это вернет обернутую ошибку 

## Упаковка ошибок
---

Когда функция которую вызывает функция получает ошибку лучше ее обернуть например так:

```Go
func someFunc(i int) (int, error) {
	j, err := funcReturningError(i)
	if err != nil {
		return 0, fmt.Errorf("wrap error: %w", err)
	}
	
	return j, nil
}

func funcReturningError(i int) (int, error) {
	if i == 0 {
		return 0, fmt.Errorf("got %d", i)
	}
	
	return i, nil
}
```

## Обработка паник
---

Отложенные вызовы ***defer*** — это код который выполняется после:
- После завершения выполнения функции.

- После выхода функции по `return` (из любого места функции).

- В случае паники (отрабатывает паника для конкретной функции, потом `defer` для неё же, потом повторяется для функции выше по стеку и так далее).

Приведение интерфейса к определенному типу

```Go 
//s - это передеваемый интерфейс 
switch s.(type){
		case nil:
			fmt.Println("it is nil!")
		case *Article:
			fmt.Println("it is a pointer!")
		case Article:
			fmt.Println("it is a structure!")
		default:
			fmt.Println("is it a bird?")
	}
```

Пустой интерфейс это любой тип данных для которого не определен набор методов 
Например:
```Go
func printInterfaceVal(v interface{}) {
  switch ____ {
  case int:
     fmt.Printf("целое число: %d\n", v)
  case string:
     fmt.Printf("строка: %s\n", v)
  case bool:
     fmt.Printf("булев тип: %t\n", v)
  }
}
```
В это функцию моет предаться любой аргумент с любым типом данных и в зависимости от его типа он выполнит определенный код
