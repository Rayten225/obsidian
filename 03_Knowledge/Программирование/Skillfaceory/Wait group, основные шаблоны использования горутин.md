
## Wait group
---

Wait group используется для синхронизации горутин. Он представляет из себя атомарный счетчик который может увеличиваться и уменьшаться. Если его значение = 0 то ждущая горутина разблокируется. 
**Добавлять счетчик для горутины нужно снаружи в не внутри иначе главный поток отработает быстрее чем запустятся горутины и процесс завершиться.**

Вот пример использования:
```Go 
wg := sync.WaitGroup //Создание переменной счетчика

for i := 1; i <= 10; i++{
	wg.Add(1) //Добавление в счетчик кол-во горутин по циклу 
	go func(){
	defer wg.Done() //Выполнение горутины и уменьшение счетчика на 1 
	//Some code
	}()
}
wg.Wait() //Ожидание обнуления счетчика (выполнение всех горутин)
```
