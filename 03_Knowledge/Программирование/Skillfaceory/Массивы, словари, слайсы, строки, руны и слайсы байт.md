## Массив
---
***Массив*** — это последовательность элементов одного типа 

***Последовательное расположение элементов*** — это особенность массивов означающая что все элементы связаны и идут друг за другом в памяти  

Массив создается так: 
```GO 
var x [5]int //Создание массива
x[4] = 100 // Присваивание элементу значение 
x := [5]float64{ 98, 93, 77, 82, 83 } //Создание массива и одновременное его заполнение
```

##### Элементы в массиве всегда индексируются с нуля

Чтобы узнать длину массива стоит воспользоваться функцией`len()`

## Цикл for range
---

```Go
for k, v := range array {
	//k — от _key_, индекс текущего элемента массива
	//v — от _value_, значение текущего элемента массива
	}
```

## Многомерный массив — массивы в массиве
---
```Go
var arr = [2][4]int{
	[4]int{1, 2, 3 ,4}
	[4]int{5, 6, 7, 8}
}
```

## Слайсы
---

***Слайс*** — это массив с динамическим определением вместимости (cap - capacity)
```Go
slice := []int{1, 2, 3}//Создание слайса 
```

## Словарь (map)
---

***Словарь (map)*** — это структура данных содержащая данные по типу ключ - значение

Ключем может быть любой тип данных кроме функции, других словарей или слайс

Инициализация словаря: 
```Go
m := make(map[string]int)
m[key] = value //Добавление записи в словарь
value, ok := m[key]//Проверка на существование элемента
delete(m, key)//Удаление элемента где m — мапа, key — ключ.
```
