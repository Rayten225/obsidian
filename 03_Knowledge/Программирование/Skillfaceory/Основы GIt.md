Основные команды для редактирования веток и репозитория:
**HEAD** - ветка на которой сейчас отслеживаются изменения

| git init                                                               | инициализация репозитория                                              |
| ---------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| git add <файлы>                                                        | Добавление файлов в commit                                             |
| git commit -m "название коммита"                                       | подготовка к отправлению в репозиторий                                 |
| git push                                                               | отправление в репозиторий                                              |
| git status                                                             | посмотреть статус коммита                                              |
| git log                                                                | посмотреть логи коммитов                                               |
| git branch <название ветки>                                            | посмотреть существующие ветки или создать новую                        |
| git merge <название ветки>                                             | слияние ветки HEAD с указанной веткой                                  |
| git fetch                                                              | загрузка изменений из удаленной ветки                                  |
| git remote (add, remove) <название репозтория> <ссылка на репозиторий> | добавление соединения с удаленным репозиторием или удаление соединения |
| git checkout <имя ветки или хэш коммита>                               | переключение между ветками репозитория или хэшами коммита из git log   |
| git reset                                                              | отмена текущего изменения ветки                                        |
| git rebase (--continue, --abort)                                       | слияние веток с сохранением историй коммитов (--adort отмена слияния)  |
| git clone <имя ветки или репозитория>                                  | клонирование удаленного репозитория или ветки                          |
