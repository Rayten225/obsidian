
## **Функции**
---

 ```Go
 func PrintHello(message string) string{
	 result := "hello," + message
	 return result
 }
```

- func — Объявление функции
- PrintHello — название 
- (message string) — передаваемые аргументы (название, тип данных)
- string{ — тип данных возвращаемого элемента
- PrintHello(message string) string — сигнатура функции

Функции могут быть аргументами другой функции:

```Go
func squareOfSums(a,b int, sumFunc func(a,b int) int) int{
	return sumFunc(a,b) * sumFunc(a,b)
}

func main() {
	sum := func(a,b int) int {
		return a+b
	}
	println(squareOfSums(3,5, sum))  // результат: 64
}
```

**Функция переданная как параметр другой функции называют *callback***  — функция обратного вызова

Чтобы передать не фиксированное количество аргументов нужно написать:

```Go 
func SumPolynomial(x float64, a ...int) int {
```

## **Структуры**
---

***Структура*** — это объект, который принимает в себя в качестве значений переменные или функции 

Пример объявления функции:
```Go 
type Circle struct {
    r float64 // radius
    x0, y0 float64 // center
}
```

Circle — название структуры
r float64 — поле значения

```Go
c2 := Circle{} //Создает экземпляр объекта 
c5 := new(Circle) //Создает экземпляр и возвращает указатель на него 
с7 := &Circle{2, 3, 4} //Создает экземпляр и возвращает указатель на него, но с инициализированными полями
```

Чтобы получить доступ к полям структуры надо воспользоваться селектором 
```Go
var c Circle
c.r = 10
println(c.x0)
```

***Композиция — это структура вложенная в структуру***

***Теги структур — это мета информация которая указывается в конце поля структуры. Поля с такими же названиями как у тегов будут записаны в соответствующие поля структуры***

***Метод структуры — это функция связанная с структурой*** 
```Go
func (c *Circle) Area() float64 { 
    return math.Pi * c.r * c.r
}
```

## Функции конструкторы
---

Функция конструктор — это функция задача которой инициализировать структуру со всеми необходимыми полями 

 **Геттеры** — это функции которые возвращают какой то элемент из структуры не давая его изменить 
 
```Go
 func (c *Circle) GetRadius() float64 {
	return c.r
}
```
 **Сеттеры** — это функции которые позволяют изменять поля в структуре 
 
```Go
func (c *Circle) SetCenter(x, y float64)  {
	if c.x0 != x {
		c.x0 = x
	}
	if c.y0 != y {
		c.y0 = y
	}
}
```
